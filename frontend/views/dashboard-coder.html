¿Qué es una variable?

Piensa en una variable como una caja etiquetada donde guardas un dato (un número, un texto, etc.). En Python, para crear una variable solo debes escribir su nombre, poner el signo igual = y asignarle un valor.

Python 
# Creamos una variable llamada 'edad' y le guardamos el número 22
edad = 22

# Creamos una variable con texto (lleva comillas)
nombre = "Valery"
